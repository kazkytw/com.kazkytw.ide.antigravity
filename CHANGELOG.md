# Changelog

All notable changes to the Antigravity Unity IDE Integration package will be documented in this file.

## [0.1.0] - 2025-11-22

### Added
- Initial release of Antigravity Unity IDE Integration
- Auto-discovery of Antigravity installations on Windows, macOS, and Linux  
- Support for opening files at specific line and column numbers
- C# project file generation for IntelliSense support
- Based on Unity Visual Studio Editor package and com.unity.ide.cursor

---

**Note**: This package was forked from Unity's Visual Studio Editor package. See the original Unity documentation for previous Visual Studio Editor history.
