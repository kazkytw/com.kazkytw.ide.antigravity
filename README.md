## How to install
- Unity -> Window -> Package Manager  
- Click "+" at the top left corner  
- Add package from git URL  
- Insert `https://github.com/kazkytw/com.kazkytw.ide.antigravity.git`  
- Add  
- Done

## How to use
After installation:
- Edit -> Preferences -> External Tools
- Select **Antigravity Editor** from the External Script Editor dropdown
- Done! Now double-clicking C# files in Unity will open them in Antigravity
